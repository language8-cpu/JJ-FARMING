# JJ Farming Website

## Project Overview
This is a multi-page informational website for **JJ Farming**, an agricultural input supplier based in South Africa. The website provides information about the organisation, its products, team, and allows visitors to make enquiries or contact the business.

## Student Information

- **Organisation Name:** JJ Farming
- **Year Established:** 2019

## Website Pages (Minimum 5 Pages)

The website consists of **5 HTML pages** as outlined in the sitemap:

| Page | Filename | Description |
|------|----------|-------------|
| Homepage | `index.html` | Hero image, brief introduction, call to action, navigation menu |
| About Us | `about.html` | Organisation's history, mission, vision, team members, goals, objectives, awards |
| Products | `products.html` | Detailed information about seeds, fertilizers, and pesticides with pricing and instructions |
| Enquiry | `enquiry.html` | Form for product enquiries, volunteering, or becoming a sponsor |
| Contact | `contact.html` | Contact information, multiple locations (3 locations), contact form, social media links |
